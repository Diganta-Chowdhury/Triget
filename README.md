# Triget
An application for making vacation reservations that helps users in researching locations, locating affordable trips and lodging. 
## Problem Identification!
1. Exotic places but no budget plan. 
2. Unreliable cost estimation. 
3. ALways  travel with hands full of cash? 
## Solutions :
1. Provides a budget plan.
2. Has a range of prices.
3. Easy decision making.
## Why to use this app?
1. Easy to use.
2. Minimum data requirement.
3. Less time consuming.
## Platforms Used!
1. IDE: Android Studio.
2. UI Framework: Flutter.
3. Language: Dart.
4. Database: SQLite.
